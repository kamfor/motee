Detailed Description

Elements values convention: 0xY000XXXX

  • XXXX : Interrupt mask (16 bits) in the Y register
  • Y : Interrupt source register (2bits)
    • 01: CR1 register
    • 10: CR2 register
    • 11: CR3 register
