# motee_v2
Highly integrated actuator for robotics

### Components
* Pololu micro DC HP motor
* Magnetic encoder
* ATtiny841 MCU
* TI high current H bridge
* RS485 controller 
* Logic Level converter 
* Current sensor

### Features
* Multi Master communication
* Easy commands
* Expansibility
* Arduino, Rbpi, AVR, ARM, PIC, PC compatibility 
* Task controlled
* High torque 
* ...
