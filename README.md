Highly integrated acutator for robotics

### Components
* Pololu micro DC HP motor
* Magnetic encoder
* ATtiny841 mcu
* TI High current H bridge
* RS485 controller 
* Ligic Level converter 
* Current sensor

### Features
* Multi Master communication
* Easy commands
* Expansibility
* Arduino, Rbpi, AVR, ARM, PIC, PC compatibile. 
* Task controlled
* High tourqe
